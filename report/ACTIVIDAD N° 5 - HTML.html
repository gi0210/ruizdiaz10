    2. Tipos de Licencia de Software

    Software libre:
    Aplicaciones disponibles para ser utilizada, copiada, modificada y distribuida por cualquiera
    GPL:
    Dirigida a Linux y limita la distribución de este software.
    Comercial:
    Desarrollado por una empresa con el objetivo de generar ganancias.
    Dominio público:
    Software sin copyright.
